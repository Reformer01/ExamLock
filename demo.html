Installation Steps:

  1. Download all extension files to a folder
  2. Open Chrome and go to chrome://extensions/
  3. Enable "Developer mode" in the top right
  4. Click "Load unpacked" and select the extension folder
  5. The extension icon should appear in your toolbar

Testing Violations:

  1. With this page open, try switching to another tab
  2. Try minimizing the browser window
  3. Press Alt+Tab or Ctrl+Tab
  4. Press F11 to toggle fullscreen
  5. Try opening developer tools (F12)
